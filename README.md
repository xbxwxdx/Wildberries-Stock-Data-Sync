### Описание программы

Эта Python-программа предназначена для автоматизации процесса сбора и сохранения данных об остатках товаров на маркетплейсе Wildberries. Она получает доступ к API Wildberries с использованием аутентификационного токена, запрашивает актуальные остатки товаров и сохраняет полученные данные в базу данных SQLite.

#### Основные шаги программы:

1. **Загрузка настроек**: Токен API загружается из переменной окружения (`os.getenv`) или задаётся вручную.
   
2. **Получение текущих данных**: Использует библиотеку `requests` для отправки GET-запроса к API Wildberries с указанием текущего дня в качестве начальной даты (`dateFrom`).

3. **Обработка JSON-данных**: Полученный ответ преобразуется в объект JSON и сохраняется в Pandas DataFrame.

4. **Создание базы данных**: Создаёт файл базы данных SQLite с именем `my_database1.db`, если таковой отсутствует, и создаёт таблицу `supplier_stocks`, если таблица ещё не существует.

5. **Запись данных**: Данные из DataFrame записываются в таблицу базы данных с помощью метода `.to_sql()` библиотеки Pandas.

6. **Закрытие соединения**: После записи данных база данных закрывается, а изменения сохраняются.

Таким образом, эта программа позволяет автоматически получать и сохранять ежедневные остатки товаров поставщика на платформе Wildberries в локальной базе данных SQLite.
